# 🖥️ Desktop Development Learning

Repositório de estudos de desenvolvimento desktop com Python e Tkinter.

## 📁 Projetos

### 🔐 Calculator
Calculadora desktop com tela de login funcional.

**Funcionalidades:**
- Tela de login com autenticação de usuário e senha
- Recursos de acessibilidade
- Calculadora com operações básicas

**Tecnologias:** Python, Tkinter, SQLite3

---

## 🛠️ Como rodar

**Pré-requisitos:** Python 3.10+

```bash
# Clone o repositório
git clone https://github.com/Gustavo-Faber/desktop-development-learning

# Entre na pasta do projeto
cd desktop-development-learning

# Rode o projeto
python calculator.py
```

> Nenhuma dependência externa pois Tkinter e SQLite3 já vêm embutidos no Python.

---

## 📚 O que estou aprendendo

- Criação de interfaces gráficas com Tkinter
- Organização de código com classes e herança
- Integração com banco de dados SQLite3
- Navegação entre telas (frames)
- Boas práticas de organização de projetos
